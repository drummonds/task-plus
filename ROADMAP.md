# Roadmap

## v0.1.0 — MVP
- [x] Interactive 12-step release workflow
- [x] Auto-detect project type (binary/library)
- [x] Auto-detect changelog format
- [x] Semver auto-increment (patch bump)
- [x] Forge release cleanup (GitHub, Codeberg, GitLab)
- [x] --dry-run and --yes flags

## Recent
- [x] `tp` alias — shorter binary name, same functionality
- [x] Documentation deployment during release (pluggable: GitHub Pages, statichost.eu)
- [x] Codeberg/Forgejo release API support (list + delete via `CODEBERG_APIKEY`)

## Future
- Pre/post release hooks
- Monorepo support
- Custom step ordering
- Additional deploy providers
